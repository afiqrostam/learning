Please be informed that we will be temporarily disabling the ISIA site from  Fri Apr 27 2018 12:30:00 GMT+0800  to allow us to migrate the existing data into the new version of ISIA.

  • Multiples photos attachment when submitting submission.

On another note, all previous generated links from the email notification are no longer  usable. Therefore, you will need to go to the site to make any update on the previous submissions.

The site is expected to be up and running again on  Wed May 2 2018 21:00:00 GMT+0800.

Apologies for all the inconvenience caused.
